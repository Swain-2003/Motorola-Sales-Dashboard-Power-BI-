# 📱 Motorola Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI Sales Dashboard** built using a Motorola mobile sales dataset. It provides insights into sales performance, customer behavior, payment methods, product popularity, and regional sales through interactive visualizations.

---

## 📊 Dashboard Features

- 📈 Total Sales KPI
- 📦 Total Quantity Sold
- 💳 Total Transactions
- 💰 Average Sales
- 🗺️ Sales by City (Map)
- 📊 Payment Method Distribution
- 📱 Top 3 Mobile Models by Quantity Sold
- ⭐ Customer Ratings Analysis
- 🏷️ Brand Filter
- 📅 Month Filter

---

## 🗂️ Dataset Information

The dataset contains the following columns:

- Transaction ID
- Day
- Month
- Year
- Day Name
- Brand
- Units Sold
- Price Per Unit
- Customer Name
- Customer Age
- City
- Payment Method
- Customer Ratings
- Mobile Model
